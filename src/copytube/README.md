# CopyTube (Main App)

This section provides the main application for this project, such as:

* PHP-FPM handling

* Views

* All client side functionality

* Routing

## Endpoints

* `GET /`

    * Renders the index view

* `GET /register`

    * Renders the view to register an account

* `POST /register`

    * Uploads an account to save into the database
